# Testing

## Frameworks
- The backend relies on `pytest` and `pytest-asyncio` according to `requirements.txt`.
- `httpx` is used for testing FastAPI endpoints.

## Coverage & Structure
- A `tests` directory exists but appears largely unpopulated locally initially (contains only `__init__.py`).
- Testing routines currently reside in standalone scripts like `test_model.py` and `colab_modular_debug.py`.
